---

## 🚀 The Architecture

CineGEN is not just a web application; it is a full-stack environment built on a robust Python/Flask backend with a highly responsive, custom-engineered UI matrix.

### Core Modules
* **The Script Studio:** An AI-powered creative sandbox utilizing the OpenRouter API to converse with the "Genie," generating cinematic lore, scripts, and production ideas in real-time.
* **The Data Workspace:** A heavy-duty analytics terminal. It features a custom ETL (Extract, Transform, Load) pipeline that ingests raw cinematic data via **Pandas/NumPy**, structures it into a secure **SQLite** database, and visualizes the financial metrics natively through an embedded **Power BI** matrix.
* **The Origin Archive:** A cinematic, scroll-triggered narrative engine built with `IntersectionObserver` that details the history and philosophy of the Ashiverse.

### Dynamic Theme Matrix
The OS features a CSS-variable architecture allowing instantaneous environment hot-swapping:
* 🌌 **Dark AMOLED:** Deep voids and electric cyan accents (Default).
* 📟 **The Matrix:** Pure hacker aesthetics with phosphor greens.
* 🏢 **Light Studio:** Pristine, high-visibility corporate mode.
* ⬛ **High Contrast:** Pure black and white for absolute utility.

## 🛠️ Tech Stack

* **Backend:** Python, Flask, Gunicorn
* **Data Engine:** Pandas, NumPy, SQLite, Microsoft Power BI
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, Phosphor Icons
* **APIs & Integrations:** OpenRouter (AI Models), TMDB (Cinematic Database)
* **Deployment:** Render (Cloud Hosting)
